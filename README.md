# YouTube-For-Smart-TV
YouTube For Smart TV — набор BAT‑скриптов и ПО для обеспечения стабильного доступа к YouTube на закрытых Smart TV‑платформах. Подходит для систем вроде Tizen, webOS, VIDAA и любых других, где нельзя установить VPN или сторонние приложения, обеспечивая корректный запуск сервиса.
